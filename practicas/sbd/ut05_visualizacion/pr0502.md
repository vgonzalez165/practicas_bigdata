# PR0502: Composición multivista en Altair. Análisis de impactos de aves


El objetivo de esta práctica será crear un panel de visualización complejo combinando múltiples vistas para analizar los accidentes aéreos provocados por fauna.

Para ello utilizaremos el dataset *birdstrikes* de Vega. Está disponible en la librería `vega_datasets`. Aségurate de convertir el campo `Flight_Date` a formato temporal para facilitar el análisis.

**NOTA**: este dataset tiene 10000 registros y si intentas representarlo con Vega Altair verás que te mostrará un error indicando que el máximo son 5000 registros. Para esta práctica puedes eliminar ese límite ejecutando la función `alt.data_transformers.disable_max_rows()`
